# Ewill

Use Vue3 to build this Website, use Scss as css preprocessor, and use Vite as build tool.

Go to see the [Demo](https://ewill.vercel.app/).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
